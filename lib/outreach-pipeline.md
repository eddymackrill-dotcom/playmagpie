# Outreach pipeline — authority building

Tracks every link-earning outreach action. You log the human actions (who you contacted,
when, outcome); I keep it current and surface the funnel. Mirrors the `lib/keyword-research.md`
discipline.

**Scope reminder:** this is the *action log*, not a link generator. Targets come from
(a) known relationships/communities (pre-populated below) and (b) the competitor backlink
recon (`lib/backlink-recon-*.md` → vetted target list). Recon tells you WHERE to aim; this
file tracks whether you actually aimed and hit. The links are earned by the human outreach +
the Transparency Report asset — not by either file existing.

## Status values
`queued` (identified, not yet contacted) · `sent` · `replied` · `won` (link live) ·
`lost` (declined / no reply after follow-up) · `n/a` (no-link channel — referral/reputation only)

- **Bitcointalk confirmed** as the one genuine community link source in the niche (already queued above).

This raises confidence in the Cloudbet/BitStarz/BC.Game outreach rows. When emailing, explicitly ask about a "featured reviews / partners / press" page (the §3 draft already does).
